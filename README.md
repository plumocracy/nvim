[FFF as fuzzyfinder](https://github.com/dmtrKovalenko/fff.nvim)

Mason for LSP

[LuaLine](https://github.com/nvim-lualine/lualine.nvim)

Snippets, Icons

[cmp as Autocomplete](https://github.com/hrsh7th/nvim-cmp)


### Dependencies
You need FFF installed for the fuzzy finder to work correctly. The instructions to do so are in the FFF readme :3
